# Milestone 1

STRATOS (STRAtegy Training and Optimization System) automates the generation, execution, and evaluation of jailbreak attacks against LLMs. This folder contains the raw and processed data collection.

---

## Dependencies

- **Python 3.10+**
- **Docling** — PDF-to-markdown conversion

```bash
pip install docling
```
If your PC has a compatible GPU (I don't) you can use PyTorch and will have to change a few lines in the code to make it work. 
I explain which lines to change in the comment at the beginning of the code, I have not tested the PyTorch version, as I am using Apple Silicon Mac as my machine. 
```bash
pip install torch (if you have a GPU)
```

---

## How to Run

All commands from the `Milestone 1/ Code` directory.

Depending on your machine, it should take a few minutes to process the PDF and convert it into MD. 

### PDF extraction

```bash
python 1_pdf_to_markdown.py "../Dataset/<paper_name>.pdf" ../Output/
```
For example:
```bash
joey@Joeys-MacBook-Pro code python 1_pdf_to_markdown.py "../Dataset/A Causal Explainable Guardrails for Large Language.pdf" ../Output/
```
**If you would like the code to run faster, change the line:**

```python
pipeline_options.do_formula_enrichment = True
```    
to:
```python
pipeline_options.do_formula_enrichment = False
```    
**By default, I have set it to False to save time when running the script**
